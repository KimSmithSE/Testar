# Tryck inte här 🟢💥

> Ett litet vibe-kodningsprojekt. Du varnas.

## Vad är det här?

En webbsida med en knapp. En enkel knapp. Du ska inte trycka på den.

Men du kommer göra det ändå.

## Vad händer om man trycker?

1. 💥 Explosionsgrafik
2. ⚡ Skärmblixt och skakeffekt
3. 🔴 Sidan byter färg till röd
4. 😏 Knappen byter text till *"Jag sa ju det"*

## Tekniker

- Vanilla HTML, CSS och JavaScript
- Canvas API för partikelexplosionen
- Inga bibliotek, inga beroenden – bara kaos

## Kör lokalt

Öppna bara `index.html` i en webbläsare. Det är allt.

## Live

Publicera via GitHub Pages: Settings → Pages → Deploy from branch → `main`
